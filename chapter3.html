3.1 Concept of Software

3.1.1 Definition of Software

Software is a collection of programs, procedures, and related data that provide instructions to a computer. Unlike hardware, which is the physical component of a computer, software is intangible and enables the computer to perform specific tasks.

3.1.2 Categories of Software

Software is broadly categorized into:

  • System Software: Includes operating systems and utilities that manage hardware resources.
  • Application Software: Programs designed for end users, such as word processors and web browsers.
  • Utility Software: Tools for system maintenance, such as antivirus software and file management utilities.
  • Web-Based Software: Applications that run via web browsers, such as Google Docs.
  • Mobile Apps: Applications specifically designed for smartphones and tablets.

3.2 Concept of Operating System

3.2.1 Introduction to Operating System

An operating system (OS) is system software that acts as an intermediary between computer hardware and the user. It manages hardware resources and provides an environment for software applications to run.

3.2.2 Role of Operating System

The operating system plays a crucial role in managing system resources such as memory, CPU, input/output devices, and file storage. It provides user interfaces, security features, and multitasking capabilities.

3.2.3 Functions of an Operating System

  • Process Management
  • Memory Management
  • File System Management
  • Device Management
  • Security and Access Control
  • User Interface Provision

3.3 Windows Operating System

3.3.1 Features of GUI-Based Operating System

Graphical User Interface (GUI) OS like Windows provides an intuitive user experience with features such as windows, icons, and menus for easy navigation.

3.3.2 Managing Files and Folders

Users can organize, create, and manipulate files using file explorers and management tools available in Windows.

3.3.3 Customization and Settings

Windows allows customization of desktop themes, backgrounds, and accessibility features.

3.4 Open Source and Mobile Operating Systems

3.4.1 Concept of Open Source Operating System

Open source OS such as Linux and UNIX allow users to modify and distribute their source code freely.

3.4.2 Types of Mobile Operating Systems

Common mobile operating systems include Android and iOS, which are optimized for touchscreen devices and provide access to various applications.
