Chapter 2: Logic Gates and Boolean Algebra

Logic Gates

Logic gates are the building blocks of digital circuits. They perform logical operations on binary inputs.

Types of Logic Gates

  • AND Gate: Outputs HIGH (1) only if both inputs are HIGH.
  • OR Gate: Outputs HIGH (1) if at least one input is HIGH.
  • NOT Gate: Inverts the input (0 becomes 1, and 1 becomes 0).
  • NAND Gate: Combination of AND followed by NOT (inverse of AND).
  • NOR Gate: Combination of OR followed by NOT (inverse of OR).
  • XOR Gate: Outputs HIGH if inputs are different.
  • XNOR Gate: Outputs HIGH if inputs are the same.

Boolean Algebra

Boolean algebra is a mathematical approach used to analyze and simplify digital circuits.

Basic Boolean Laws

  • Identity Law: A + 0 = A, A * 1 = A
  • Null Law: A + 1 = 1, A * 0 = 0
  • Idempotent Law: A + A = A, A * A = A
  • Complement Law: A + A' = 1, A * A' = 0
  • Distributive Law: A * (B + C) = A * B + A * C
  • Absorption Law: A + (A * B) = A
