2.1 Number System and Conversion

The number system is a way of representing numbers in different formats. Common number systems include decimal, binary, octal, and hexadecimal. Understanding number systems is essential in computing as computers operate in binary.

2.1.1 Decimal, Binary, Octal, and Hexadecimal Number System

Decimal (Base-10): The standard number system used by humans, consisting of digits 0-9.

Binary (Base-2): The fundamental system used by computers, consisting of only 0 and 1.

Octal (Base-8): Uses digits from 0 to 7 and is sometimes used in computing.

Hexadecimal (Base-16): Uses digits 0-9 and letters A-F, commonly used in programming and memory addressing.

2.1.2 Number System Conversion

Conversions between number systems are crucial in computing:

  • Decimal to Binary: Divide the number by 2 and record the remainders in reverse order.
  • Binary to Decimal: Multiply each binary digit by powers of 2 and sum the results.
  • Binary to Octal: Group binary digits in sets of three and convert.
  • Binary to Hexadecimal: Group binary digits in sets of four and convert.

2.1.3 Binary Addition and Subtraction

Binary arithmetic operations include:

  • Binary Addition: Follows similar rules as decimal addition but only uses 0 and 1.
  • Binary Subtraction: Uses borrowing like decimal subtraction.

2.1.4 One’s and Two’s Complement Methods

One’s complement is obtained by flipping all bits, while two’s complement is obtained by adding 1 to the one’s complement.

2.2 Logic Functions and Boolean Algebra

Boolean algebra is used in digital circuits and computing. It includes variables that take values 0 or 1.

2.2.1 Boolean Values, Truth Table, Boolean Expression

Boolean expressions use AND, OR, and NOT operations. A truth table represents all possible inputs and outputs.

2.2.2 Logic Gates

Logic gates perform Boolean operations:

  • AND: Outputs 1 if both inputs are 1.
  • OR: Outputs 1 if at least one input is 1.
  • NOT: Inverts the input.
  • NAND, NOR, XOR, XNOR: Variants of the above gates with specific logic.
