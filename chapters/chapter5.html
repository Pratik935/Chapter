5.1 Programming Concept

5.1.1 Introduction to Programming Languages

Programming languages are formal languages used to communicate instructions to a computer. They are essential for creating software, applications, and websites.

5.1.2 Low Level, High Level, 4GL Programming Languages

Low-level languages like Assembly interact closely with hardware, whereas high-level languages like Python, Java, and C++ provide abstraction for easier programming. Fourth-generation languages (4GL) such as SQL are designed for specific tasks like database management.

5.1.3 Compiler, Interpreter, and Assembler

Compilers translate an entire program into machine code before execution, while interpreters translate and execute code line-by-line. Assemblers convert assembly language into machine code.

5.1.4 Syntax, Semantic, and Runtime Errors

Syntax errors occur due to incorrect code structure, semantic errors involve logic mistakes, and runtime errors occur during execution due to invalid operations.

5.1.5 Control Structures: Sequence, Selection, and Iteration

Control structures dictate the flow of a program. Sequence executes instructions linearly, selection (if-else, switch) makes decisions, and iteration (loops) repeats processes.

5.1.6 Program Design Tools â€“ Algorithm, Flowchart, and Pseudocode

Algorithms define step-by-step logic for problem-solving. Flowcharts visually represent algorithms using symbols, while pseudocode expresses logic in human-readable format.

5.1.7 Absolute Binary, BCD, ASCII, and Unicode

These encoding schemes represent data in binary form for storage and communication. ASCII and Unicode handle text characters, while BCD represents decimal numbers in binary.

5.2 C Programming Language

5.2.1 Introduction and Features of C Language

C is a powerful, structured programming language used for system and application development. It is known for efficiency, portability, and flexibility.

5.2.2 Structure of a C Program

A C program consists of a header, main function, and statements. Each program follows a structured format for readability and execution.

5.2.3 C Preprocessor and Header Files

The C preprocessor processes source code before compilation, and header files contain reusable function definitions and macros.

5.2.4 Character Set Used in C

C recognizes letters, digits, special characters, and whitespace as part of its character set for defining variables, functions, and operations.

5.2.5 Use of Comments

Comments in C improve code readability and documentation. They can be single-line (//) or multi-line (/* */).

5.2.6 Identifiers, Keywords, and Tokens

Identifiers are user-defined names for variables, functions, and arrays. Keywords are reserved words like int, return, and while.

5.2.7 Basic Data Types in C

C has fundamental data types such as int, float, char, and double, which define variable types and memory requirements.

5.2.8 Constants and Variables

Constants hold fixed values, while variables store data that can change during program execution.

5.2.9 Type Specifiers

Type specifiers such as signed, unsigned, short, and long modify data type properties.

5.2.10 Simple and Compound Statements

Simple statements execute a single operation, whereas compound statements group multiple instructions using curly braces {}.

5.2.11 Operators and Expressions

C includes arithmetic (+, -), relational (==, !=), logical (&&, ||), assignment (=), unary, and conditional operators.

5.2.12 Input/Output (I/O) Functions

Common I/O functions in C are printf() for output and scanf() for input.

5.2.13 Selection Control Statements

Selection statements include if, if-else, nested if, and switch-case constructs.

5.2.14 Iteration Control Statements

Looping structures in C include while, do-while, and for loops.

5.2.15 Arrays

Arrays store multiple values of the same data type. One-dimensional and two-dimensional arrays are commonly used.

5.2.16 Strings and String Functions

Strings store sequences of characters. Common string functions in C include strlen(), strcat(), strcmp(), strrev(), strcpy(), strlwr(), and strupr().


Some Example C Programs.

Program to input a number and check if it is prime or not.

                
                    #include <stdio.h>

                    int main() {
                        int num;
                        printf("Enter a number: ");
                        scanf("%d", &num);
                        for(int i = 2; i < num; i++) {
                            if (num % i == 0) {
                                printf("Not a Prime Number.\n");
                                goto end;
                            }
                        }
                        printf("Prime Number.\n");
                        end:
                        return 0;
                    }

Program to input a number and find it's factorial.

                
                    #include <stdio.h>

                    int main() {
                    	int num, factorial = 1;
                    	printf("Enter a number: ");
                    	scanf("%d", &num);
                    	for(int i = num; i > 0; i--) {
                    		factorial *= i;
                    	}
                    	printf("The factorial of %d is %d.\n", num, factorial);
                    	return 0;
                    }

Program to input a number and check if it is palindrome or not.

                
                    #include <stdio.h>

                    int main() {
                    	int num, temp, rev = 0;
                    	printf("Enter a number: ");
                    	scanf("%d", &num);
                    	temp = num;
                    	while(temp != 0) {
                    		rev = rev * 10 + temp % 10;
                    		temp = temp / 10;
                    	}
                    	if (rev == num) {printf("Palindrome.\n"); }
                    	else { printf("Not a palindrome.\n"); }
                    	return 0;
                    }

Program to generate fibonacci series upto n terms.

                
                    #include <stdio.h>

                    int main() {
                    	int a = 0, b = 1, c = 0, num;
                    	printf("Enter a number: ");
                    	scanf("%d", &num);
                    	if (num == 1 ) { printf("0\n"); }
                    	else if (num == 2 ) { printf(" 0 1\n"); }
                    	else {
                    		printf("%d %d ", a, b);
                    		for (int i = 3; i <= num; i++ ) {
                    			c = a + b;
                    			printf("%d ", c);
                    			a = b, b = c;
                    		}
                    	}
                    	return 0;
                    }

Program to input a number and check it is perfect number or not.

                
                    #include <stdio.h>

                    int main() {
                    	int num, sum = 0;
                    	printf("Enter a number: ");
                    	scanf("%d", &num);
                    	for(int i = 1; i < num; i++ ) {
                    		if(num % i == 0) { sum += i; }
                    	}
                    	if(sum == num) { printf("Perfect Number.\n"); }
                    	else { printf("Not a perfect Number.\n"); }
                    	return 0;
                    }
