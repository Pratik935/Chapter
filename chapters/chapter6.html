Chapter 6: Web Technology - I

Web technology refers to the tools and techniques used to create and manage websites and web applications.

Introduction to Web Technology

  • Internet: A global network that connects millions of computers.
  • World Wide Web (WWW): A system of interlinked web pages accessed via the internet.
  • Web Browsers: Software applications used to access and view web pages (e.g., Google Chrome, Mozilla Firefox).
  • Web Servers: Computers that store and serve web pages to clients.
  • Web Pages: Documents written in HTML that can be viewed in a web browser.

Introduction to HTML

HTML (HyperText Markup Language) is the standard language used to create web pages.

  • HTML Elements: Basic building blocks of a web page.
  • Tags: Used to define different elements (e.g., <h1>, <p>, <a>).
  • Attributes: Provide additional information about an element (e.g., href for links).

CSS Basics

CSS (Cascading Style Sheets) is used to style HTML elements and improve the appearance of web pages.

  • Selectors: Define which HTML elements to style.
  • Properties: Define the appearance (e.g., color, font-size).
  • External CSS: A separate file used for styling multiple pages.

Introduction to Static and Dynamic Websites

  • Static Websites: Contain fixed content that does not change dynamically.
  • Dynamic Websites: Generate content dynamically using programming languages (e.g., PHP, JavaScript).
