6.1 Introduction to Web Development

Web development refers to the process of creating and maintaining websites. It includes web design, front-end and back-end development, and web hosting. The web has become an essential medium for sharing information, conducting business, and social networking.

6.2 Web Browsers and Search Engines

Web browsers are software applications used to access and navigate the internet. Popular web browsers include Google Chrome, Mozilla Firefox, Microsoft Edge, and Safari. Search engines like Google, Bing, and Yahoo help users find relevant content on the web.

6.4 HTML: The Language of the Web

6.4.1 Structure of HTML

HTML (Hypertext Markup Language) is the standard language for creating web pages. It structures web content using elements like headings, paragraphs, lists, and links.

6.4.2 Basic HTML Tags

HTML elements include tags like <h1> for headings, <p> for paragraphs, <a> for links, and <img> for images.

6.5 Cascading Style Sheets (CSS)

6.5.1 Introduction to CSS

CSS (Cascading Style Sheets) is used to style and format HTML documents. It controls the layout, colors, fonts, and responsiveness of a webpage.

6.5.2 Types of CSS

  • Inline CSS: Applied directly to an HTML element.
  • Internal CSS: Defined within a <style> tag in an HTML document.
  • External CSS: Stored in a separate file and linked to the HTML document.
